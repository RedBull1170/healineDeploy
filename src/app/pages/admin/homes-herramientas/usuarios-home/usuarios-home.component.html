Usuarios

Esta sección permite gestionar los usuarios del sistema.
Aquí podrás consultar, registrar, actualizar y eliminar usuarios
según sea necesario. Mantén un control preciso sobre los
datos de acceso y la información relevante de cada usuario.

Consultar Usuario

Accede a los detalles de un usuario registrado en el sistema.

Registro De Usuario

Registra nuevos usuarios con información relevante.

Editar Usuarios

Actualiza o elimina los datos de un usuario existente.
